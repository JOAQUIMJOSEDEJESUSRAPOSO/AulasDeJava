# tipos primitivos

### numeros inteiros  ⟶ byte, short, int, long
- byte => 8 bits => -128 a 127; ⟶ para usar esse tipo o numero deve está dentro dessa faixa de intervalo
- short => 16 bits => -32.768 a 32 767; ⟶ para usar esse tipo o numero deve está dentro dessa faixa de intervalo
- int => 32 bits => -2.147.483.648 a 2.147.483.647; ⟶ para usar esse tipo o numero deve está dentro dessa faixa de intervalo
- long => 64 bits => -9.223.372.036.854.775.808 a 9.223.372.036.854.775.807; ⟶ para usar esse tipo o numero deve está dentro dessa faixa de intervalo

### numeros decimais ⟶ float, double
- float => 32 bits => numero de precisão simples => 32.4, 23.45, 12,60;
- double => 64 bits => precisão dupla;

### Strings ⟶ string, char
- String => representar palavras e frases; ⟶ String usa aspas duplas

- char
    char => representar um unico caractere => a, b, c; ⟶ char usa aspas simples

### booleanos
- boolean => true or false

# Loops
- Existem varias formas de declarar um loop
- uma das principais é o ``for()``
- outra forma tambem muito utilizada é o ``while()``

## Diferenças principais do for() para while()

| **FOR**                                                                    | **WHILE**                                                |
| -------------------------------------------------------------------------- | -------------------------------------------------------- |
| Usado quando você **sabe quantas vezes** quer repetir                      | Usado quando você **não sabe quantas vezes** vai repetir |
| A variável de contagem, condição e incremento ficam **todos no cabeçalho** | Só a **condição** fica no cabeçalho                      |
| Melhor para **loops contados**                                             | Melhor para **loops condicionais**                       |
